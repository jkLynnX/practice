# Master Document Checklist - 2025 Taxes

Check off each item as you gather it. Save documents to the appropriate `docs/` subfolder.

---

## Income Documents

- [ ] **W-2** from employer — _Get from: employer payroll portal or HR_
  - Save to: `docs/w2/`
- [ ] **1099-INT** (bank interest) — _Get from: each bank's website, usually under Tax Documents_
- [ ] **1099-DIV** (dividends) — _Get from: brokerage/investment account, if applicable_
- [ ] **1099-G** (government payments/refunds) — _Get from: state agency, if applicable_
- [ ] **Any other 1099s** — _Get from: each issuer_

## Mortgage & Home Documents

- [ ] **Form 1098** (Mortgage Interest Statement) — _Get from: mortgage lender's website under Tax Documents_
  - Save to: `docs/1098/`
- [ ] **Property tax statement - PRIMARY HOME** — _Get from: county tax assessor website_
  - Save to: `docs/property-tax/`
- [ ] **Homeowners insurance declaration page** — _Get from: insurance company portal (needed for home office calc)_
  - Save to: `docs/home-office/`

## Tree Farm (Schedule F) Documents

- [ ] **Property tax statement - FARM LAND** — _Get from: county tax assessor website_
  - Save to: `docs/property-tax/`
- [ ] **Texas Ag Exemption documentation** — _Get from: county appraisal district or your files_
  - Save to: `docs/farm-expenses/`
- [ ] **Equipment & supply receipts** — _Get from: Home Depot, Lowe's, Tractor Supply, etc. (see Receipt Collection Guide)_
  - Save to: `docs/farm-expenses/`
- [ ] **Seed/seedling/plant purchase receipts** — _Get from: nurseries, suppliers_
  - Save to: `docs/farm-expenses/`
- [ ] **Fertilizer & chemical receipts** — _Get from: store accounts or credit card statements_
  - Save to: `docs/farm-expenses/`
- [ ] **Contractor/labor invoices** — _Get from: your records, email, or request from contractors_
  - Save to: `docs/farm-expenses/`
- [ ] **Did you pay any single contractor $600+?** If yes, you must issue them a **1099-NEC**
  - [ ] 1099-NEC filed for contractor: ________________
  - [ ] 1099-NEC filed for contractor: ________________
- [ ] **Vehicle mileage log for farm trips** — _Create from: calendar, Google Maps history, or start logging now_
  - Save to: `docs/farm-expenses/`
- [ ] **Farm insurance** (if any) — _Get from: insurance provider_
  - Save to: `docs/farm-expenses/`
- [ ] **Farm property improvement receipts** (fencing, sheds, barns, irrigation, land clearing) — _Get from: store accounts, contractors_
  - Save to: `docs/farm-expenses/`
- [ ] **Fuel receipts for farm equipment** — _Get from: gas station receipts or credit card statements_
  - Save to: `docs/farm-expenses/`

## Home Office Documents

- [ ] **Room measurements taken** — Office sq ft: _____ | Total home sq ft: _____
- [ ] **Utility bills for 2025** (electric, gas, water, internet) — _Get from: each utility provider's website_
  - Save to: `docs/home-office/`
- [ ] **Home repair/maintenance receipts** (that apply to whole house) — _Get from: store accounts, contractors_
  - Save to: `docs/home-office/`

## Personal Home Improvement Receipts (Cost Basis Tracking)

> Note: These are NOT deductible now, but increase your home's cost basis for when you sell.

- [ ] **Home Depot receipts** — _Get from: homedepot.com account (see Receipt Collection Guide)_
  - Save to: `docs/home-improvements/`
- [ ] **Lowe's receipts** — _Get from: lowes.com account (see Receipt Collection Guide)_
  - Save to: `docs/home-improvements/`
- [ ] **Other home improvement receipts** — _Get from: various stores/contractors_
  - Save to: `docs/home-improvements/`

## LynnX LLC

- [ ] **Texas franchise tax report filed** — _File at: Texas Comptroller website (even if dormant, "No Tax Due" report required)_
- [ ] **LLC formation documents** (for reference) — _Get from: your files_
  - Save to: `docs/other/`

## Other Tax Documents

- [ ] **Prior year tax return (2024)** — _Get from: TurboTax account (download PDF)_
  - Save to: `docs/other/`
- [ ] **Health insurance form** (1095-A, 1095-B, or 1095-C) — _Get from: employer or insurance marketplace_
  - Save to: `docs/other/`
- [ ] **Charitable donation receipts** — _Get from: charities, church, organizations_
  - Save to: `docs/other/`
- [ ] **Student loan interest** (Form 1098-E) — _Get from: loan servicer, if applicable_
  - Save to: `docs/other/`
- [ ] **IRA contribution records** — _Get from: brokerage account, if applicable_
  - Save to: `docs/other/`
- [ ] **Estimated tax payments made** — _Get from: IRS account online, if applicable_
  - Save to: `docs/other/`
- [ ] **Social Security number** (have handy for filing)
- [ ] **Bank account & routing number** (for direct deposit of refund)
