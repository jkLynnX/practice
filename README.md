## Document Storage

Save your downloaded/scanned documents into these folders:

```
docs/
  w2/                  - W-2 forms from employer
  1098/                - Mortgage interest statements
  farm-expenses/       - Farm receipts, contractor invoices
  home-office/         - Home office related receipts
  home-improvements/   - Personal home improvement receipts (for cost basis)
  property-tax/        - Property tax statements (home + farm land)
  other/               - Any other tax documents
```

## How to Use This

1. Start with **01-MASTER-CHECKLIST.md** - work through it top to bottom, gathering each document
2. Read **02-TAX-STRATEGY.md** to understand the game plan
3. Follow **03-RECEIPT-COLLECTION-GUIDE.md** to pull all your receipts
4. Fill in **05-HOME-OFFICE-WORKSHEET.md** and **06-SCHEDULE-F-FARM-WORKSHEET.md** with your numbers
5. Open TurboTax and follow **04-TURBOTAX-WALKTHROUGH.md** step by step

> **Disclaimer:** This is an organizational tool, not professional tax advice. Consider consulting a CPA, especially regarding farm loss deductions and hobby loss rules.
